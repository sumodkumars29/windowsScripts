To update a new git directory to an existing git repo
Option 1:Allow unrelated histories (Most common)
		git pull origin main --allow-unrelated-histories
		
Option 2: If you want to keep your local history as primary (Recommended)
	
	# Fetch the remote changes
		git fetch origin

	# Merge allowing unrelated histories
		git merge origin/main --allow-unrelated-histories

	# Resolve any conflicts, then add and commit
		git add .
		git commit -m "Merge unrelated histories"

	# Push to remote
		git push -u origin main

Option 3: If you want to overwrite remote with your local content

	# Force push your local branch to overwrite remote
		git push -u origin main --force

Option 4: If you want to start fresh (discard remote content)

	# Fetch remote changes
		git fetch origin

	# Reset your branch to match remote (will lose local changes)
		git reset --hard origin/main

	# Then pull any remaining changes
		git pull origin main
